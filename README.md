# Econometrics-Research-Report

This project investigates key macroeconomic relationships in the Australian economy using time series econometric techniques. The analysis focuses on inflation, unemployment, monetary policy, exchange rates, terms of trade, and US economic activity through ARDL and Vector Autoregression (VAR) models.

The project was completed as part of an Applied Econometrics course and demonstrates practical applications of modern macroeconometric modelling.
